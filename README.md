# Описание платежного модуля ИнвойсБокс для AmiroCMS 6.0+

Платёжный модуль для интеграции платёжной системы «ИнвойсБокс» и CMS AmiroCMS версии 6.х. Реализована поддержка платёжного API.

## Установка модуля

1. Распакуйте архив в корень вашего сайта;
2. В панели управления сайтом перейдите в раздел <strong>"Сервис" -> "Способы оплаты"</strong>;
3. В списке оплат найдите "invoicebox" и нажмите кнопку <strong>"Установить"</strong>.

## Настройка модуля

Перейдите в раздел <strong>"Сервис" -> "Способы оплаты"</strong> панели администрирования.
Найдите в списке оплат "invoicebox", нажмите кнопку <strong>"Редактировать"</strong>.

Укажите необходиме данные:

- Идентификатор магазина (ID магазина)
- Региональный код магазина
- Ключ безопасности (API ключ)

### Специфические настройки для Amiro CMS

Кроме базовой интеграции вы можете установить специфические настройки:

1. Процент надбавки к сумме заказа - устанавливается значение в процетах, к сумме заказа в зависимости от выбранной платежной системы;
2. Постоянная надбавка к сумме заказа - фиксированная сумма добавляющаяся к сумме заказа в зависимости от выбранной платежной системы;
3. Валюта надбавки - валюта в которой будет добавляться надбавка к сумме платежа;
4. Тестовый режим - включите его для проведения тестовых платежей, при включении этого режима, вы пройдете все шаги в платежном терминале InvoiceBox, но деньги с вашей карты списаны не будут.

## Настройка языковых переменных

Вероятно вам потребуется указать дополнительные языковые переменные для корректного вывода платёжного модуля на сайте.
Это можно сделать следующим образом:

1. Зайдите в раздел <strong>Шаблоны модулей::локализации</strong>;
2. В форме <strong>"Поиск по шаблону"</strong> укажите <strong>eshop_purchase</strong>, откройте его и вставте в конец файла следующие языковые переменные:

``` %%pay_with_invoicebox_option%en%%
Pay with InvoiceBox
%%pay_with_invoicebox_option%ru%%
Оплатить через ИнвойсБокс

%%pay_with_invoicebox%en%%
Pay with InvoiceBox
%%pay_with_invoicebox%ru%%
Оплатить через ИнвойсБокс
```
